## Overview

Advanced algebra focuses on structure: factor clever expressions, transform
systems, and use symmetry. This is an introduction to the techniques and tools that will be used in the upcoming topics and to make make sure there is a clear well understanding and not a sudden jump

## Key Ideas
 - Using clever substitution to reduce casework and to avoid brute-force
 - Identifying where factorisation can be utilised (which usually gives away the solution)
 - Having a close eye and identifying patterns along with doing smart grouping

## Core Skills

### Use Symmetric Substitution

If an expression is symmetric in $x$ and $y$, set $s=x+y$ and $p=xy$ to reduce
degree. This is a theorem which will be covered later. This is especially useful in question sof the recurring theme where they want the value of an expression which has the roots as the variables, and the roots are of a certain quadriatic equation.

### Factor Strategically

Check for standard identities (difference of squares, sum/difference of cubes, Sophie-Germain identity...)
before expanding. This allows for factorisation, which frequently happens to reveal most of the solution.

### Complete the Square

Rewriting quadratics as $(x-h)^2+k$ helps with bounds and root structure. This allows us to use difference of squares which again as mentioned before, will lead to factorisation and basic number theory skills can be used for the solution.

## Worked Example

Solve $x+y=5$ and $x^2+y^2=13$.

Compute $2xy=(x+y)^2-(x^2+y^2)=25-13=12$, so $xy=6$.
Then $t^2-5t+6=0$(Using Vieta's relations, which will be introduced later), giving $t=2$ or $3$. So $(x,y)$ is $(2,3)$ or $(3,2)$.

## More Examples

### Example 1: Vieta and Symmetry

If $x+y=4$ and $xy=1$, find $x^3+y^3$.

$x^3+y^3=(x+y)^3-3xy(x+y)=64-12=52$.

### Example 2: Completing the Square

Find the minimum of $x^2-6x+13$.

$x^2-6x+13=(x-3)^2+4$, so the minimum is $4$.

### Example 3: System Reduction

Solve $x+y=6$ and $xy=5$.

$t^2-6t+5=0$ gives $t=1$ or $5$, so $(x,y)=(1,5)$ or $(5,1)$.

**Note**: The above examples are simply for a feel on the problems and make sure to not be overly concerned with the theorems themselves

## Strategy Checklist

- Look for symmetry and use $s=x+y$, $p=xy$.
- Factor before expanding.
- Complete the square for extrema or root structure.
- Identify opportunities for regrouping
- Expanding expressions that factor cleanly.
- Dropping solutions when converting to $s$ and $p$.
- Forgetting to check back in the original system.
